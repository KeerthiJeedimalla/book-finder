# Book Finder

A responsive web application that helps users discover books using the Open Library API. Built with React and Tailwind CSS.

## Live Demo

[View Live Application](https://your-book-finder-url.vercel.app) 
*(Replace with your actual Vercel URL after deployment)*

## Features

- **Dual Search Mode** - Search by book title or author name
- **Book Covers Display** - Show book covers from Open Library
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Real-time Search** - Instant results with loading states
- **Error Handling** - User-friendly error messages for better experience
- **Clean Modern UI** - Beautiful interface with Tailwind CSS
- **Keyboard Support** - Press Enter to search

## Technology Stack

- **Frontend Framework**: React 18
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **API**: Open Library Search API
- **Icons**: Emoji-based icons
- **Deployment**: Vercel
